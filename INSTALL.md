# Install zurg-cli

- Set the directory to be in your install of `zurg-cli`. The recommended directory house Zurg is `$HOME/` on Mac.
- Execute `$ sh install.sh`
- Quit your terminal and reset the session.


# Uninstall

- Zurg can be uninstalled by running `$ sh uninstall.sh` in the `zurg-cli` directory.
- Quit your terminal and reset the session.
